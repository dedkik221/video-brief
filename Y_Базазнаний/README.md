# Media Team Knowledge Base + Brief (Standalone MVP)

Этот проект — отдельный сайт для медиакоманды:
- модуль `База знаний` (процессы, инструкции, шаблоны),
- модуль `Быстрый заказ услуг` (мастер-заявка без интеграций на этапе демо).

## Цели этапа
- Показать рабочий standalone-сайт без Jira/SSO/PDF интеграций.
- Зафиксировать структуру модулей так, чтобы можно было безопасно развивать фичи независимо.

## Быстрый старт
1. Откройте `/site/index.html` в браузере.
2. Перейдите в разделы:
- База знаний: `/site/knowledge/montage-process.html`
- Мастер заявки: `/site/request/new-request.html`

## Структура
- `docs/` — план, границы модулей, подготовка к интеграции.
- `site/` — статический демо-сайт.
- `site/knowledge/` — статьи базы знаний.
- `site/request/` — формы заявки (пока локальная логика).
- `site/js/` — изолированная JS-логика фич.
- `site/data/` — справочные данные для форм.
