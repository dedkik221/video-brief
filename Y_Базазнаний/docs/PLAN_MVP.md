# План MVP (Standalone, без интеграций)

## Этап 1: Демо-версия
1. Каркас сайта: главная, база знаний, мастер заявки.
2. 1-2 реальные страницы процессов (начинаем с монтажа).
3. Валидируемый мастер заявки (локально, без отправки наружу).
4. Демонстрация стейкхолдерам.

## Этап 2: Стабилизация
1. Доработка полей по обратной связи.
2. Финализация контент-шаблонов для ролей.
3. Подготовка контрактов интеграции.

## Этап 3: Интеграции (после согласования)
1. Jira API (создание задачи).
2. Генерация PDF из заявки.
3. Прикрепление PDF к задаче.
4. Корпоративная авторизация.

## Что не делаем в MVP
- Прямую интеграцию с Jira.
- Автогенерацию PDF в прод-режиме.
- Сложную аналитику и отчеты.
