# Подготовка к интеграциям (после демо)

## Будущие точки интеграции
1. Jira: создание задачи по заполненной форме.
2. PDF: рендер финальной заявки в единый документ.
3. Jira Attachment: прикрепление PDF к задаче.
4. SSO: доступ через корпоративную авторизацию.

## Что должно быть готово заранее
- Схема данных заявки (`serviceType`, `eventDate`, `location`, `techNeeds`, `notes`).
- Список обязательных полей по каждому типу услуги.
- Правила именования задачи в Jira.
- Макет PDF-шаблона (поля и секции).

## Контракт (черновик)
- Вход мастера: UI-данные формы.
- Выход мастера: JSON-объект заявки.
- Следующий этап: JSON -> Jira + PDF.
