## 2) Ключевые изменения в форме

- Убрали пошаговый "Дальше/Назад" формат в пользу вертикальной структуры.
- Добавили явные статусы заполнения блоков и верхний индикатор прогресса.
- Сохранили локальное автосохранение (LocalStorage) с переключателем.

### Типы роликов

Оставлен выбор **одного** типа ролика (это важно):

- Отчётный (highlight/recap)
- Производство / процессы / люди
- Тематический контент
- Реклама продукта
- Вертикальный контент (Reels/Shorts)
- Нестандартный заказ

Примечание: множественный выбор типа ролика отменён по решению пользователя.

### Поля и формулировки

- `CTA` сохранён в понятном виде: **"Что зритель должен сделать дальше"**.
- Удалены `KPI 1 / KPI 2`.
- `Аудитория + уровень` объединены в одно поле:
  - **"Целевая аудитория ролика"**
  - с понятным placeholder для неэкспертных пользователей.
- Для тезисов добавлена поясняющая инструкция и пример.
- Блок ответственного за правки переработан в:
  - **ЛПР по правкам**
  - ФИО
  - Тип контакта (Телефон / Telegram / Почта)
  - Контакт

  - Горизонтальный (16:9)
